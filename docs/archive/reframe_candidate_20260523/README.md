# Reframe Candidate Archive

This folder preserves the reframe candidate layout as a reference snapshot.

Included:

- `dist_html/`: built `_reframe.html` entry files from `dist/`
- `src_views/`: source Vue folders for the reframe pages and shared reframe styles
- `docs/`: reframe description document snapshot

The active design direction is to keep the base portal pages and absorb only the useful reframe information hierarchy into them.
